[小ネタ][WIP] Universal Media ServerからJPLAYStreamerでm4aファイルを聞く

Jan 1, 0001

Minim serverでJPLAYStreamerでm4aファイルが「aac:wav24」を追加してあるにもかかわらず、聞けない状況でどうしたもんかと思いUniversal Media Serverから試してみました。
　新規に作成したJPLAYStreamer.confの内容はこんな感じ。

>UserAgentSearch : UpnpDetailsSearch : JPLAY , JPLAYStreamerWrapEncodedAudioIntoPCM : trueTranscodeFastStart : trueTranscodeAudio : WAVTranscodeExtensions : flac,m4a,mp3,alacSupported : f:wav n:2 s:192000 m:audio/L24


　m4aファイルは聞けるようになりましたが、どの種別のファイルも途中で再生が止まっている状況。またFLACの変換でビット深度が24bitにもかかわらず16bitで送信されてしまう。mp3ファイルに至っては再生すら始まらず。
　お知恵がございましたら拝借したく思います。
