KENWOOD MG-E504を買った

Jan 1, 0001



ヨドバシ川崎で買った。14800円のポイント10%だったかな。色はホワイト。

MEDIA kegということで期待はしていました・・・。ええ、想像以上に音がいいんだろうな、と。
で、音ですが、14800円ならアリかな、という感じです。まあ、携帯電話のプレーヤーよりはよかったですが・・・。
操作性はいいです。説明書読まなくても使えます。
あとWindowsのエクスプローラーでドラッグ&ドロップして曲を登録できるのがいいですね。Windows Media Playerでもできるらしいですが。再生の方もフォルダ指定で再生できるのもいいです。海外プレーヤのいいところを持ってきてる。

あと連続再生時間が長いのがいいですね。MP3の128kbpsを再生するなら54時間もちます。

あまりオススメではありませんが「日本のメーカの安いプレーヤほしい！」という人なら候補に入れてもいいんじゃないでしょうか。あともうじきこれのメモリ容量が2GBのが出ますが、値段は2000円程度しか違わないので4GBのこっちの方がいいと思います。
