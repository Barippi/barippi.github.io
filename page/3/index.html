[小ネタ] コカコーラの自動販売機に持ってた誤解が解けた話

　ども、明日は出勤日のばりっぴです。今日は自動販売機の話をします。
　コカコーラの自動販売機、かざしたICカードを自動判別してくれて便利ですよね。こういうやつ。

<a target:”_blank” title:”DSC_0206” href=”/images/5decae94.jpg”><img class:”pict” hspace:”5” alt:”DSC_0206” border:”0” height:”853” width:”480” src=”/images/5decae94.jpg”/>

　ちょっと反射して私が写ってるのはどうでもいいとして、欠点として「複数の支払い方法を入れたスマホで混乱して結果買えない」という欠点がある、と思ってました。昨日まで。
　昨日、弊社の社長がこのタイプの自販機で飲み物を買うときにこの電子マネーのパネルを押したんです。「えっ」と思って。そしたらマニュアル操作でどの支払い方法にするか選べたんですね。実はこのパネル、ボタンでもあったのです。これUIで損してる。自動判別するけど、せっかく選べるんだからそういうのが直感的に分かるようになって欲しいです。自動判別できるという利点の方を大きく出したくてこうなったんだろうけれども…。このパネルじゃ押せる、と思わない。少なくとも私は。私だけですかねぇ…。

セプトクルールのメモ帳をオススメしていきたい

　ども。夜更かしばりっぴです。
　今日はseptcouleur(セプトクルール)のメモ帳をオススメしていきたいと思います。
　特にA7サイズのが胸ポケットにちょうど入ってジャストサイズです(？)。
　こんなの。

<a target:”_blank” title:”DSC_0195” href=”/images/e3dbbf82.jpg”><img class:”pict” hspace:”5” alt:”DSC_0195” border:”0” height:”270” width:”480” src=”/images/e3dbbf82.jpg”/>

　おっしゃれー(棒)。これを1冊1ヶ月強で使い切ります。書き心地もなかなか良いです。なかなかメモを取る癖が付かないという方はこういう形から入っていくのもアリなのではないかと思います。
　公式サイトは<a target:”_blank” href=”https://www.e-maruman.co.jp/products/detail.php?product_id:2671">こちら。そう、マルマンの文具なのです。お値段もおやすめ。合わせるボールペンはジェットストリームが良いかと思います。アフィリエイトで貼り付けたジェットストリームは筐体の色使いがダサいので、実際お店に行って選ぶのがいいかと思います。例えば、

<a target:”_blank” title:”DSC_0199” href=”/images/069925a2.jpg”><img class:”pict” hspace:”5” alt:”DSC_0199” border:”0” height:”853” width:”480” src=”/images/069925a2.jpg”/>

な感じので、これは元々全部黒インクだったのですが左から青、赤、黒のインクを入れています。こういうライフハックもできるジェットストリーム、オススメです。長期間残さなきゃいけない書類以外ならばフリクションもいいのですが、フリクションはにじみやすいという難点がありまして…。そういう面でもジェットストリームとかサラサあたりがオススメラインナップとなっております。

[小ネタ] アイドルマスターシャイニーカラーズは運ゲーなのか

　結論としては「ほぼ運ゲーじゃなくできる」です。

　最初はまぐれでWING優勝できたりして「ほーん」みたいな感じになるのですが、やっていくに連れてプレーヤーのスキルが上昇すると運の要素をできる限り排除できる、まるでアケマスのようなゲームになってきます。
　っていうのを約1年、方々眺めてみて感じたことです。「今の若い子にこんな難しいゲーム与えていいのだろうか」とも思いましたが、先達の書くWikiなど、集合知をぐんぐん吸収してSランクアイドルをバンバン作ったりしてて「今の若い子やるな」と思ってます。デジタルネイティブだと情報の集め方とその咀嚼の仕方が上手い。

マクセルの1000円イヤホンを買って聴いてみました

　どうも、Windows Insider MVPになれなかったばりっぴです。
　ドラッグショップという体をしたなんでも売ってるクリエイトSDにてマクセルの1000円イヤホン、MXH-C110を買ってきたので聴いてみました。
　付属のイヤーピースのままだと低音～中音あたりがよく出てて、その辺に自信なのかな、とかなんか音が遠いなと思いつつJVCのスパイラルドットイヤーピースに変えたら化けましたね。高音もでるじゃないあなた。音も近くなるじゃないあなた。
　なんつーか、イヤホンよりもイヤーピースの方が高くなりますが、1セット持っておくといいですね、スパイラルドットイヤーピース。うちにある1000円イヤホンシリーズ(セブンイレブンPB、パナソニック)、どの子もスパイラルドットイヤーピースで化けます。それくらいこの子達はポテンシャルは持ってます。ただ、買ったのが白い筐体なのでスパイラルドットイヤーピースはつけるとかっこ悪い。買うなら黒ですね。ぜひにー。
