まーだジョルテ使ってるんですか！？これからはOne Calendarの時代でしょ！？

長いことジョルテを使ってまいりましたが、この度AndroidのカレンダーアプリをOne Calendarに変更しました。なぜかというと、Android版があると分かったからです。

そもそもWindowsストアアプリで出ていたOne Calendarですが、ちょっとバグがあると☆2とか1をつけがちな日本人が軒並み☆4、5という高評価をつけていて「それならば」とWindowsストアアプリをダウンロードしたのが始まり。

  • Windows Live
  • Outlook.com
  • Googleカレンダー
  • Windows Exchange
  • Office 365
  • Facebookの誕生日とかのイベント
  • iCloud
  • Webcal
  • CalDAV

に対応していて、Windows版、Android版で同じエクスペリエンス(iPhone版もあるけどそちらは持ってないので試せない)。特にWindows版だとスタート画面にピン止めするとライブタイルでその日以降の予定が見られるのが素晴らしい。

ジョルテに課金はしなかったが、One Calendarには速攻で課金してしまった。そのくらいWindows8.1以降と相性がいいアプリです。

ジョルテに飽きてきたなーという方は是非。

小ネタ AIRismの扱いを間違えると逆に臭くなる話

AIRism、防臭効果のために買ったんだけどここ数週間、とても臭かった。

なんでだろうと思って「AIRism 寿命」でググったら「柔軟剤を使用すると機能が低下し、戻りません」みたいなことを書いてあるところを見つけて、「これかー」となりました。

逆に雑菌の増殖する環境になっててそれで臭かったらしい。あとは皮脂汚れが残りやすいので漂白剤使うといいとも書いてあった。たしか酸素系漂白剤。塩素系は生地にダメらしい。

昨日ユニクロに寄って、パンツ買って今日新しいの履いていったら臭わない！人権を取り戻した感じになりました。まる。

[小ネタ] コカコーラの自動販売機に持ってた誤解が解けた話

　ども、明日は出勤日のばりっぴです。今日は自動販売機の話をします。
　コカコーラの自動販売機、かざしたICカードを自動判別してくれて便利ですよね。こういうやつ。

<a target:”_blank” title:”DSC_0206” href=”/images/5decae94.jpg”><img class:”pict” hspace:”5” alt:”DSC_0206” border:”0” height:”853” width:”480” src=”/images/5decae94.jpg”/>

　ちょっと反射して私が写ってるのはどうでもいいとして、欠点として「複数の支払い方法を入れたスマホで混乱して結果買えない」という欠点がある、と思ってました。昨日まで。
　昨日、弊社の社長がこのタイプの自販機で飲み物を買うときにこの電子マネーのパネルを押したんです。「えっ」と思って。そしたらマニュアル操作でどの支払い方法にするか選べたんですね。実はこのパネル、ボタンでもあったのです。これUIで損してる。自動判別するけど、せっかく選べるんだからそういうのが直感的に分かるようになって欲しいです。自動判別できるという利点の方を大きく出したくてこうなったんだろうけれども…。このパネルじゃ押せる、と思わない。少なくとも私は。私だけですかねぇ…。

[小ネタ] アイドルマスターシャイニーカラーズは運ゲーなのか

　結論としては「ほぼ運ゲーじゃなくできる」です。

　最初はまぐれでWING優勝できたりして「ほーん」みたいな感じになるのですが、やっていくに連れてプレーヤーのスキルが上昇すると運の要素をできる限り排除できる、まるでアケマスのようなゲームになってきます。
　っていうのを約1年、方々眺めてみて感じたことです。「今の若い子にこんな難しいゲーム与えていいのだろうか」とも思いましたが、先達の書くWikiなど、集合知をぐんぐん吸収してSランクアイドルをバンバン作ったりしてて「今の若い子やるな」と思ってます。デジタルネイティブだと情報の集め方とその咀嚼の仕方が上手い。
